# Requerimientos de Huerto Connect

## Requerimientos funcionales
1. El sistema debe permitir el registro de usuarios.
2. El sistema debe permitir el inicio de sesión.
3. El sistema debe permitir registrar huertos agrícolas.
4. El sistema debe permitir editar y eliminar huertos.
5. El sistema debe permitir subir imágenes de plantas para análisis.
6. El sistema debe generar recomendaciones mediante Inteligencia Artificial.
7. El sistema debe incluir un chatbot de asistencia.
8. El sistema debe permitir gestionar regiones agrícolas.

## Requerimientos no funcionales
1. La interfaz debe ser intuitiva.
2. El sistema debe ser escalable.
3. La información debe almacenarse de forma segura.
4. El sistema debe responder de manera rápida.
